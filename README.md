## Conjure

Automatic contract checking for your RSpec suite

### Requirements

  * Rspec 3.0.0.beta2

### Getting Started

Install the rspec-contracts gem alongside rspec in your gemfile.

    group :development, :test do
      gem "rspec", "3.0.0.beta2"
      gem "rspec-contracts"
    end

Then update your bundle.

    $ bundle
